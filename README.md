The ECE repository contains all the scripts related to ECE data processing across the IFF footprint.

1) MI_ECE_ProviderList_Nov2024.R
  Processing steps for cleaning the input datasets and integrating them to produce the final
  MI ECE Provider list based on November 2024 data.

2) Detroit_ECE_GapAnalysis_Nov2024_v2.R
  Final version of the Nov 2024 Detroit ECE gap analysis
  Streamlined script for the Nov 2024 Detroit ECE gap analysis. 
  Wrote final excel file with tract, neighborhood, zcta and council district level 
  ECE gap, quality gap, supply, quality supply and demand and 
  two geodatabase files - one with supply data and one without.
  
3) Detroit_ECE_GapAnalysis_2015revised.R
  Final version of the revised 2015 Detroit ECE gap analysis
  Streamlined script corresponding with the Nov 2024 Detroit ECE gap analysis. 
  Included provider count, quality provider category and 
  wrote final excel file with tract, neighborhood, zcta and council district level 
  ECE gap, supply and demand and two geodatabase files - one with supply data and one without.
